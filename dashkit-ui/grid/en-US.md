# Grid

A grid system based on the flex display property, which refer to [Flexbox Grid](http://flexboxgrid.com/).

<Demos />

## API
### Grid
Property    | Description    | Type      | Default   |
|---------- |-------- |---------- |-------------  |-------- |
| className | className of the component | - | - |
| fluid     | is fluid                   | boolean | false |

### Grid.Row
Property    | Description    | Type      | Default   |
|---------- |-------- |---------- |-------------  |-------- |
| className | className of the component | - | - |
| xs        |  grid size  | number  | -  |

### Grid.Col
Property    | Description    | Type      | Default   |
|---------- |-------- |---------- |-------------  |-------- |
| className | className of the component | - | - |
